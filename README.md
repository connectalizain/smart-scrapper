# Smart Scrapper

Smart Scrapper is a data scraping tool designed to extract business information from YellowPages.ca. It provides a simple web interface to initiate scraping and view results.

## Current Features

*   **Web Scraping:** Scrapes business listings from any YellowPages.ca search result URL.
*   **Data Extraction:** Extracts the following information for each business:
    *   Business Name
    *   Phone Number
    *   Website
*   **Interactive UI:** A simple web interface built with [Chainlit](https://chainlit.io/) that allows you to paste a YellowPages.ca URL and receive the scraped data in a table format.
*   **Powered by AI:** Uses a Gemini model to understand the user's request and execute the scraping tool.

## How to Use

1.  Run the application.
2.  Open the web interface.
3.  Paste a YellowPages.ca search result URL into the chat window.
4.  The tool will scrape the data and display it in a table.

## Future Ambitions

The long-term vision for Smart Scrapper is to evolve it into an autonomous lead generation platform. This will involve:

*   **Autonomous Workflow:** Creating a system that can not only scrape data but also act on it.
*   **Lead Generation:** Using the scraped phone numbers to initiate contact with potential leads through automated calls or text messages.
